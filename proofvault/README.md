# ProofVault

Build professional portfolios with privacy-respecting analytics. Showcase your work with confidence and measure its impact without compromising visitor privacy.

## 🚀 Features

- **Portfolio Builder**: Create stunning portfolios with our intuitive section-based editor
- **Project Management**: Organize your work with projects, media, and technology tags
- **Media Library**: Upload and manage images and documents with Cloudflare R2 storage
- **Privacy-First Analytics**: Understand your portfolio's performance without tracking individual visitors
- **Responsive Design**: Looks great on all devices
- **SEO Optimized**: Built-in Open Graph and meta tags for perfect link previews
- **Accessibility**: WCAG 2.2 AA compliant for inclusive access

## ✨ Highlights

- **Lightning Fast**: LCP < 2.5s, Lighthouse scores ≥ 95
- **Secure**: Row Level Security, rate limiting, and security best practices
- **Easy to Use**: Go from signup to published portfolio in minutes
- **Privacy Focused**: Analytics that respect visitor privacy
- **Customizable**: Multiple themes and layout options

## 🛠 Tech Stack

- **Frontend**: Next.js (App Router), React, TypeScript, Tailwind CSS
- **Backend**: Next.js Route Handlers
- **Database**: Supabase PostgreSQL
- **ORM**: Prisma
- **Authentication**: Supabase Auth (Magic Link, Google OAuth, GitHub OAuth)
- **Storage**: Cloudflare R2
- **Deployment**: Koyeb
- **Email**: Resend
- **Analytics**: PostHog
- **Monitoring**: Sentry
